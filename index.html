    How to make AdLib Stories:

    Enter your text into the "Your Story" box using parentheses to denote your data selections (what would be the blank spots in a paper Mad Lib).

    Currently, fields are limited to the "Available Fields" list, but we plan on adding an option for users to add their own on the fly.


    How it works:

    Each field denotes a field from a dataset hosted at the WPRDC. When parsing your selections, queries are made through the WPRDC's RESTFul API to pull a list of values for that field. A random value is then selected and inserted into your story. Check out the code GitHub!
